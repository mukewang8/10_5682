<h3>课程介绍：</h3>
<p>编写一个<a title="查看与 区块链 相关的文章" target="_blank">区块链</a>，并基于区块链实现一个类似比特币的加密货币，实现区块链与比特币最核心的几个功能，如区块链整体结构、挖矿生成新区块、共识机制、比特币交易、基于密码学的交易验证、比特币钱包、区块链去中心化网络、P2P网络通讯等，以期让大家对区块链底层原理有一个清晰的理解，为深入学习区块链开发打下坚实的基础。</p>
